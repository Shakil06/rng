# Documentation Directory

This directory contains comprehensive documentation for the TRNG project.

## Files

### EEE458_ProjectReport_Group27.pdf

Detailed project report including:

- **Introduction**: Overview of TRNGs and motivation
- **Design Methodology**: Detailed explanation of the architecture
- **Implementation**: Hardware design and implementation details
- **Chaotic XOR Network**: Analysis of the non-linear feedback system
- **ISU Network**: Description of the interleaved shuffle units
- **Statistical Testing**: Wald-Wolfowitz Runs Test results and analysis
- **Entropy Analysis**: Shannon entropy calculations
- **Performance Evaluation**: Throughput, resource usage, and quality metrics
- **Results**: Experimental results and validation
- **Conclusions**: Summary and future work
- **References**: Academic and technical references
